# middle-frontend
study with Result School

## Установка

1. Клонировать репозиторий https://github.com/catfishsoup69/middle-frontend
2. Установить зависимости
```
npm i
```

### Запуск в режиме разработки
```
npm start
```
### Билд проекта в режиме разработки
```
npm run build:dev
```
### Продакшн билд проекта
```
npm run build
```
